== Usage ==


1. Install and Enable json-api plugin
2. Install and Enable jp-custom-post-type plugin
	2.1 Create a Movie-reviews content
3. Install and Enable jp-custom-post-type-json-api plugin 	
	3.1 you should see the json object using this link: <your wordpress url>?json=moviereview/get_moviereviews
